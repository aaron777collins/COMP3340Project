Description:

Funstuff.site is an e-commerce website that specializes in drop-shipping cool and interesting products from Aliexpress. The reason we chose this model is because it involves 0 capital and minimal maintenance. It also requires very little commitment from our team as we won't have any stock to sell off.

We will use an Azure VPS to run the server and database. We are also using many new tools like React, Typescript, NodeJS, etc. Trello will be used to coordinate our team's activities. It has many automation rules to make our lives easier.

Tools Used

ReactJS, TypeScript, Material UI,VSCode, NodeJs, Npm, MySQL, Docker, Azure VPS
